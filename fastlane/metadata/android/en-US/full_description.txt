<p><i>Atomic – Periodic Table</i> is an open-source app, designed to be intuitive, which means that the app works great for both those who only wants basic chemistry and physics information like an elements atomic weight to more advanced users who might want more detailed data like an isotope’s halftime or an elements ionization energy. There are multiple tables available in the app as well as some options to easily visualize data with more coming in the near future. ’Atomic - Periodic Table’ is a burden-less experience with no ads or other nonsense, just all the data you need for your chemistry or physics project!</p><p><i>Atomic - Periodic Table</i> will also continue to grow as more feature updates will be coming bi-monthly adding more data sets, additional details, additional visualization options for the main ’Periodic Table’ and much more.</p><p><br><b>Atomic - Highlights</b></p><ul><li>Periodic Table – Bring an intuitive periodic table with you on the go</li><li>Dynamic Table – The main table can switch data with a simple tap of a button to easily show and visualize important data.</li><li>Electronegativity Table – Easily see the difference of electronegativity between different elements.</li><li>Solubility Table – See which compounds are soluble with which</li><li>Electrochemical Series – Easily view elements electrode potential</li><li>Dictionary – Master the periodic table with the help of an inbuilt dictionary</li><li>Element Details – Information about every element</li><li>Favorite Bar – Select which element details are most important to you, for easier access</li><li>Offline Mode – To save data enable the offline mode which will disable the loading of some images.</li><li>And much more!</li><li>Isotope Panel – Explore a vast number of isotopes</li></ul><p><b>Current Data:</b></p><ul><li>Atomic Number</li><li>Atomic Weight</li><li>Discovery Details</li><li>Group</li><li>Appearance</li><li>Isotope Data</li><li>Density</li><li>Electronegativity</li><li>Block</li><li>Electron Shell details</li><li>Boiling Point (Kelvin, Celsius and Fahrenheit)</li><li>Melting Point (Kelvin, Celsius and Fahrenheit)</li><li>Electron Configuration</li><li>Ion Charge</li><li>Ionization Energies</li><li>Atomic Radius (Empirical and calculated)</li><li>Covalent Radius</li><li>Van Der Waals Radius</li><li>Phase (STP)</li><li>Protons</li><li>Neutrons</li><li>Isotope Mass</li><li>Halflife</li><li>Fusion Heat</li><li>Specific Heat Capacity</li><li>Vaporization Heat</li><li>And more</li></ul>
